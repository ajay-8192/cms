# CMS Backend

## tool for content management

### BE local setup

Make sure that local redis server is up and running

Do it by installing redis and then in linux running the command <code>sudo service redis-server start</code>

1. NVM install
2. Node version 22.2.0
3. Install packages
4. For production mode
   1. start (`npm start`)
5. For development mode
   1. run as dev (`npm run start:local`)

### For Local BE

1. Run backend application in local mode (`node server.js --local`)
